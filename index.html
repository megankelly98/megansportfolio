Welcome to Megan's Portfolio

Welcome to my portfolio website! My name is Megan Kelly. On this website you will find out all about me, a lot of my portfolio work, information about me and how to get in touch! The work I have on this website includes my photography work, design work and digital artwork! You will find these under the portfolio page.

The work you see on my website is work I have done as a part of my college course but also some that I did for fun and others I have done for friends.

I am currently available for freelance work so if my work, so browse through my site if that's why you're here. If my work interests you don't hesitate to get in touch via my contact page, there you will find a form to fill out. I can also answer any questions you have.
